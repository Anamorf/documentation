stream.publish

$description
Метод публикует заметку в ленте пользователя

$params#uid
Идентификатор пользователя. Укажите uid при вызове этого метода без ключа сессии.

$params#target_id


$params#message
Сообщение, которое пользователь вводит для заметки во время публикации. Максимальная длина составляет 100 символов.

$params#attachment
JSON-кодированный объект, содержащий текст публикации, связанные ссылки, тип содержимого (image, mp3, flash), а также любые другие пары ключей/значений, которые могут потребоваться

$params#action_links
JSON-кодированный массив объектов ссылок действия, содержащих текст ссылки и гиперссылку.
